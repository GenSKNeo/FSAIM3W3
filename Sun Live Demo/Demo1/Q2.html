My Learning Journal – Using HTML Tags and Attributes

During this module, I explored several HTML tags and their attributes. These tags help structure content effectively, while attributes improve usability and accessibility.

To dive deeper into web development, I often visit MDN Web Docs for HTML references.

Tags I Have Practiced:

  • <p> – Paragraph
  • <a href=""> – Hyperlink
  • <img src=""> – Image
  • <ul>, <li> – Unordered List
  • <ol> – Ordered List

Steps to Create a Web Page:

  1. Start with the <!DOCTYPE html> declaration
  2. Use the <html>, <head>, and <body> structure
  3. Add headings, text, images, and links
  4. Apply CSS for styling
  5. Use JavaScript to make it interactive
